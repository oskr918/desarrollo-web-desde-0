Lección 1: Introducción a HTML

HTML, que significa "Lenguaje de Marcado de Hipertexto" (por sus siglas en inglés), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es la estructura básica que define la información en una página y cómo se presenta al usuario.

Conceptos básicos

En esta lección, exploraremos algunos conceptos fundamentales de HTML:

  • Etiquetas: Las etiquetas son elementos fundamentales en HTML y se utilizan para estructurar el contenido. Por ejemplo, la etiqueta <p> se utiliza para definir párrafos.
  • Atributos: Los atributos proporcionan información adicional sobre las etiquetas. Por ejemplo, la etiqueta <img> puede tener un atributo "src" que especifica la fuente de la imagen.
  • Estructura básica: Todo documento HTML comienza con la etiqueta <html> y tiene secciones como <head> y <body>.

Práctica

¡Ahora es tu turno! Crea un documento HTML simple utilizando lo que has aprendido en esta lección. Puedes utilizar un editor de texto como Visual studio code o cualquier entorno de desarrollo web.

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<title>Mi Página HTML</title>
</head>
<body>
<h1>¡Hola, mundo!</h1>
<p>Este es un ejemplo de una página HTML.</p>
</body>
</html>

Lección 2: Estructura de una Página HTML

En la lección anterior, aprendiste los conceptos básicos de HTML. Ahora, nos centraremos en la estructura general de una página HTML y cómo organizar el contenido de manera efectiva.

Estructura Básica

Una página HTML consta de varias secciones clave:

  • <!DOCTYPE html>: Declara la versión de HTML que estás utilizando.
  • <html>: Contenedor principal de todo el documento HTML.
  • <head>: Contiene información meta sobre el documento, como el título y enlaces a hojas de estilo.
  • <body>: Contiene el contenido real de la página, como texto, imágenes, enlaces y más.

Elementos y Etiquetas

En HTML, utilizamos diferentes elementos y etiquetas para estructurar el contenido:

  • Encabezados: <h1>, <h2>, ..., <h6> para títulos y subtítulos.
  • Párrafos: <p> para texto normal.
  • Listas: <ul>, <ol>, <li> para listas no ordenadas y ordenadas.
  • Enlaces: <a> para crear enlaces a otras páginas.
  • Imágenes: <img> para mostrar imágenes.

Práctica

¡Ahora es el momento de aplicar lo que has aprendido! Crea una página HTML simple que incluya títulos, párrafos, listas y al menos una imagen.
